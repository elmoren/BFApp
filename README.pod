=head1 Boundary Finder Web App

=head2 SYNOPSIS

   morbo BoundaryFinder

Then open web browser to localhost:3000

=head2 OVERVIEW

The BFApp is a simple web interface for the boundary-finder-lib built with Mojolicious. Users can run blasts, run the boundary finder on the blast hits, or just run the boundary finder on any subject sequence.

It also provides a simple sequence search tool for the searching of small amino acid sequences in a nucleotide sequence. This is provided for searching for sequences too small to use tblastn.
